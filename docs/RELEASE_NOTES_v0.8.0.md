# OpenDAV v0.8.0 Release Notes

Welcome to OpenDAV v0.8.0! This massive update focuses on overhauling the telemetry track map into a highly interactive, dynamic, and visually stunning racing tool, alongside premium UI polish and data export functionality.

## What's New

### Advanced Track Map Visualization
- **Dynamic Auto-Follow:** A new toggle that smoothly locks the camera to track the car's position in real-time during playback, maintaining your preferred zoom level.
- **Heading-Up Auto-Rotate:** A powerful new playback mode that continuously rotates the track map so the camera is always oriented behind the car, providing an immersive onboard perspective.
- **Smooth Camera Rotation & Compass:** We've added an interactive compass widget to the bottom right of the track map. Simply click and drag to smoothly orbit and rotate your view of the track to any angle.
- **Reference Ghost Dots & Gap Tethers:** When comparing laps, your White and Cyan reference laps now appear as physical dots on the track map, dynamically synchronized to their relative elapsed time. We also implemented a dashed "Rubber Band" that tethers the active car to the reference cars, making physical track gaps instantly obvious.
- **The Magnifier (Exaggerated Divergence):** Introduced a slider that mathematically magnifies lateral racing line differences up to 20x, causing tiny deviations to visibly diverge on the map.

### Premium Polish & Utilities
- **Dark Obsidian Splash Screen:** Replaced the generic loading window with a high-fidelity image-backed splash screen featuring a sleek dark obsidian overlay for a truly premium launch experience.
- **One-Click CSV Export:** Added a lightweight, single-click CSV export system natively into the dashboard header. You can now instantly dump full IBT telemetry sessions alongside their header metadata directly to disk for use in external tools.

## Improvements & Fixes
- **Visual Consistency:** Synchronized reference track map lines to match the 2.0 pixel thickness of the active lap for better visual clarity.
- **Cleaned UI Elements:** Removed unnecessary emojis from UI buttons for a sharper, more professional aesthetic.
