# OpenDav v0.3.0 Release Notes

Welcome to OpenDav v0.3.0! This release focuses heavily on upgrading the Track Map tab into a powerful, interactive coaching tool. We've completely overhauled how sector times and deltas are visualized to give you immediate, intuitive feedback on entry and exit performance.

## What's New

### Interactive Track Map Sector Deltas
We've replaced the old floating text labels with sleek, integrated info boxes attached directly to the apex of every corner on the track map. 
- **Delta Mode:** When comparing against a Cyan or White reference lap, the box at each turn instantly shows the time gained (Green) or lost (Red) through that specific corner.
- **Raw Time Mode:** If no reference lap is active, the boxes automatically switch to displaying the absolute raw time spent navigating that corner in a clean, semi-transparent neutral box.

### Enhanced Visual Precision & Contrast
When hunting for subtle differences in a driving line, absolute precision is key.
- **OpenDav Orange Base Lap:** The primary active lap is now drawn in the signature OpenDav Orange by default (when not color-coded by sector). This guarantees stark contrast when comparing against Cyan or White reference laps.
- **Thinner Track Lines:** We've slimmed down all track lines to reduce overlap and allow you to see exactly where the tires are placed. Active laps have been reduced from `3.0` to `2.0` width, and reference laps from `1.8` to `1.2`.
