# v0.5.0 - Revamped Splashscreen + New graphing features

**v0.5.0** brings many new features and fixes that will help drivers improve. New features have been implemented that will be specifically useful for coaching as well as fast data review/debriefing after sessions. 

## Whats new

### Brand & UI Revamp
- **New Splash Screen**: Implemented a sleek new animated splash screen featuring the brand new square and full-text OpenDav logos that seamlessly crossfade on launch.
- **Refreshed Sidebar & App Icon**: The application now sports the transparent lightweight logo variant in the sidebar header and a fresh application window icon.
- **Revamped User Interface**: Adjusted colors, layouts, and scaling across the app for a cleaner, more premium feel.

### Enhanced Telemetry Graphs
- **Revamped Graphing Legends**: The legends for the time series charts have been completely redesigned for better readability, and now correctly compute `base - reference` delta readings, allowing you to instantly see where you are gaining or losing speed.
- **Track Sections on Time Series**: The time series line graphs now intuitively display track sections/corners in the background, bridging the gap between track position and raw telemetry data.
- **Intelligent Chart Shading**: Added dynamic background shading to the line charts that highlights where the base lap is faster (green) or slower (orange) than the reference lap on a sector-by-sector basis.

*Enjoy the new look and the deeper insights provided by the enhanced charting system! We're laying the foundation for exciting new data integration modules next.*
